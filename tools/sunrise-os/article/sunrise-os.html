Sunrise OS (32-bit)

Equipment

  1. /reboot - Instant reboot of the ESP32 board.
  2. /reboot (ns, nm, nh) - Reboot after a certain amount of time:
    • s - seconds
    • m - minutes
    • h - hours
  3. /sleep - Switch to sleep mode.
  4. /wkup - Exit sleep mode.

Accounts and Users

When you first turn on the system, an account with Administrator rights will be automatically created, for subsequent accounts Guest rights will be the default .

  1. /createacc - creating a new account.
  2. /listacc- viewing all previously created accounts with indication of rights, name and password (temporarily).
  3. /rights (admin or guest) (account name)- issuing or changing rights for a specific account, optionally Administrator or Guest .

Control - read equipment

  1. /command1 $$ /command2... - sequential execution of commands from left to right, the quantity and speed are limited by the power limitations of your board.
  2. /rominfo - display information about the occupied and free permanent memory of the device.
  3. /raminfo - display information about the occupied and free RAM of the device.
  4. /boardinfo - display of information about the processor frequency, memory (RAM, ROM) "according to the passport".
  5. /maxperf - maximum performance mode, meaning ESP32 operation at maximum frequency (240 MHz).
  6. /balperf - balanced performance mode, reducing the device frequency to 160 MHz.
  7. /enersav - power saving mode, frequency reduction to 80 MHz (WI-FI functions will not be available below 80).
  8. /uptime - display of ESP32 operating time from the start.

Network

  1. /checkwifi - a simple test of the quality of the connection between the board and the router via Wi-Fi.
  2. /conninf - deeper network test, displaying IPv4, subnet mask, gateway, MAC address, signal strength in dBm, SSID and BSSID, and encryption type.
  3. /listnwork - a list of available nearby WI-FI networks, as well as displaying the presence of a password.
  4. /connwork (название сети) - attempt to connect to the selected network, as well as analysis of protection for entering a password, if available.
  5. /ping - test connection speed with google.com.
  6. /ping (example.com) - test the connection speed with the site you have chosen.

Files and folders

  1. /checkfs - starts checking user data files and file system integrity (SPIFFS).
  2. /fs - view all available files on the device.
  3. /makng (file name) - creating a text file (.txt) with the selected name.
  4. /dellfs (file name) - delete the file you selected.
  5. /rdfs (file name) - reading the selected file.
  6. /rnmfs (old name) (new name) - rename the selected file.
  7. /rdction (file name) (contents to be stored) - editing the contents of the selected file (overwriting the contents completely with the new ones).
  8. /conhtml (file name.txt) - convert the selected .TXT file to .HTML file. Select a file with the .txt extension only.
  9. /contxt (file name.html) - convert the selected .HTML file to a .TXT file. Select a file with the .html extension only.
  10. /delfold (/folder name) or /delfold (/folder1/folder2...) - deleting the selected folder in the root directory or along the selected path.
  11. /delfile (file name.txt) - deleting the selected file in the root directory or along the selected path.
